**Description for the MinhThanh**
=========================

Link via: https://usevia.app/

*Layout 1*

Fn + F1 = Toggle mute

Fn + F2 = Volume Up

Fn + F3 = Volume Down

Fn + Tab = Switch mode

Fn + Backspace = Toggle on/off RGB

Fn + \ = Switch RGB

Fn + 1 = Bluetooth 1

Fn + 2 = Bluetooth 2

Fn + 3 = Bluetooth 3
